# 🚀 GLM Code Skills 包

这是一个为 GLM Code 用户预装的专业技能包，包含 10 个精心设计的写作和视频创作技能。

## 📦 技能清单

### 🥇 核心技能 (P0 - 必装)
1. **AI味审校** - 降低AI检测率，增加人味
2. **图片配图与上传** - 自动生成文章配图
3. **个人素材库搜索** - 搜索真实经历案例

### 🥈 高价值技能 (P1 - 推荐)
4. **视频封标与承接检查** - MrBeast策略优化
5. **视频脚本口语化审校** - 让脚本更适合录制
6. **Prompt分类保存** - 自动分类管理prompts
7. **选题生成** - 快速生成文章选题

### 🥉 辅助技能 (P2 - 可选)
8. **长文转X内容** - 文章转社交媒体内容
9. **视频大纲生成** - 生成视频脚本方案
10. **信息搜索与知识管理** - 新产品技术搜索

## 🎯 安装方式

这些技能会在 GLM Code 首次安装时自动预装到用户目录：
```
~/.claude/skills/
```

## 💡 使用方法

安装完成后，用户可以直接使用相关技能，无需额外配置。Claude 会根据用户的任务描述自动识别并加载相应的技能。

例如：
- "帮我审校这篇文章，让它更像人写的" → 自动加载 AI味审校
- "给这篇文章配个图" → 自动加载 图片配图与上传
- "帮我生成一个视频脚本" → 自动加载 视频大纲生成
